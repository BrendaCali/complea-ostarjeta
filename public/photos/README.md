# Carpeta de Fotos

Sube aquí la foto del cumpleañero.

## Instrucciones:

1. El archivo debe llamarse exactamente: `cumpleanero.jpg` (o .png/.jpeg)
2. Súbelo a esta carpeta `/public/photos/`
3. La foto aparecerá en un círculo encima de la cabeza del león en la segunda imagen

**Nota:** El código usa `cumpleanero.jpg` como nombre. Si usas otro nombre, actualiza el código en `/src/app/App.tsx`.
